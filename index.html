What It Checks

README

Verifies a README file exists so new developers can understand the project.

LICENSE

Checks for a LICENSE file so ownership and usage terms are clear.

SECURITY.md

Looks for a security policy so vulnerability reports have a clear path.

Dependency Lockfiles

Detects Node.js, Python, Rust, Go, Ruby, PHP, and .NET dependencies and checks for lockfiles.

File Tree Hygiene

Scans for excessive depth, clutter files, and sensitive-looking filenames.

Handoff Score

Produces a 0-10 score and band rating for handoff readiness.

Use as a GitHub Action

- uses: xcapselx/repo-health-checks@v1 with: path: '.' fail-on-low-score: 'false' output-format: 'markdown'

Runs on every push, PR, or weekly schedule. Uploads a health report artifact.
