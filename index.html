What It Checks

README

Verifies a README exists so new devs understand the project.

LICENSE

Checks for a LICENSE file so ownership and usage are clear.

SECURITY.md

Looks for a security policy for vulnerability reports.

CI Workflows

Checks for GitHub Actions or other CI configuration.

Test Structure

Looks for test directories and test file patterns.

Dependencies

Detects dependency manifests and lockfiles.

.gitignore

Checks .gitignore covers common build artifacts.

Community

Checks for CONTRIBUTING, CODE_OF_CONDUCT, templates.

File Tree

Scans for depth, clutter, and sensitive filenames.

Use as a GitHub Action

- uses: xcapselx/repo-health-checks@v2 with: path: '.' fail-on-low-score: 'false'
