# Start Here

Use this file if you cloned the repo and want the shortest path through the free material.

## 1. Run the Free Checklist

Open [README Health Mini Checklist](README_HEALTH_MINI_CHECKLIST.md) and score your README in about 10 minutes.

Use `yes`, `partial`, or `no` for the checklist questions, then score the six areas from `0` to `2`.

## 2. Compare Against the Sample

Read the [Sample Repo Health Snapshot](SAMPLE_REPO_HEALTH_SNAPSHOT.md) and [Sample Scorecard](SAMPLE_REPO_HEALTH_SCORECARD.md) to see what a finished repo-health note can look like.

## 3. Choose the Next Step

- If you only need the quick pass, use the free checklist and stop there.
- If you want deeper worksheets and a report structure, use the self-serve Repo Health Snapshot Kit: https://designsy1.gumroad.com/l/toksiz
- If you want scoped help on a specific repo, open a done-for-you intake issue: https://github.com/xcapselx/repo-health-checks/issues/new?template=repo-health-check.yml

The self-serve kit is a downloadable worksheet/report structure. It does not include a custom repo review.

## Optional Signal

If the free checklist was useful, you can share a short checklist result:

https://github.com/xcapselx/repo-health-checks/issues/new?template=checklist-result.yml

This is optional. Do not include sensitive access details, customer data, private service details, or production access details.
